# StartAgro Service Documents

Desktop alkalmazás szerviz munkalapok feldolgozására.

## Funkciók

- PDF indexelés
- Fotók párosítása
- SQLite adatbázis
- PDF előnézet
- Gyors keresés

## Technológia

Tauri

React

TypeScript

SQLite

Material UI

pdf.js

## Indítás

npm install

npm run tauri dev

## Build

npm run tauri build
